## Demo and Documentation
### Usage
Insert this preloader.js in document `<head>` or `<body>` with "async" attribute

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Preloader demo</title>

	<script async src="../preloader.js"></script>
</head>
<body>
```
