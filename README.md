## Usage
Insert this preloader.js in document `<head>` or `<body>`

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Preloader demo</title>

	<script src="../preloader.js"></script>
</head>
<body>
```
